## Types of First-Order ODEs

| Type                   | Standard Form                 | Technique                                                                                                   |
| :--------------------- | :---------------------------- | :---------------------------------------------------------------------------------------------------------- |
| **Separable**          | $p(y)y' = q(x)$               | Separate the variables and integrate.                                                                      |
| **First-order linear** | $y' + p(x)y = q(x)$           | Rewrite as $\frac{d}{dx}(I \cdot y) = I \cdot q(x)$, where $I = e^{\int p(x)dx}$, and integrate with respect to x. |
| **First-order homogeneous** | $y' = f(x, y)$ where $f(tx, ty) = f(x, y)$ | Change variables: $y = xV(x)$, and reduce to a separable equation.                                         |
| **Bernoulli**          | $y' + p(x)y = q(x)y^n$        | Divide by $y^n$ and make the change of variables $u = y^{1-n}$. This reduces the differential equation to a linear equation. |
| **Exact**              | $M dx + N dy = 0$, with $M_y = N_x$ | The solution is $\phi(x, y) = c$, where $\phi$ is determined by integrating $\phi_x = M, \phi_y = N$.           |

*(Note: The formula for $M \times (N)^{-1}$ seems unrelated to the typical classification of ODEs shown and appears to be a separate, potentially incorrect, or highly specialized formula for a different context.)*

## Linear Second-Order and Higher-Order ODEs

### General Form:
$y'' + p(x)y' + q(x)y = r(x)$

### Homogenous Form:
$y'' + p(x)y' + q(x)y = 0$

### General Solution:
$y = C_1y_1 + C_2y_2 + y_p$
where $y_1$ and $y_2$ are linearly independent solutions.

### Homogenous Solution, $y_h$
If $p(x), q(x)$ are constants, for $y'' + ay' + by = r(x)$:
*   **Characteristic Homogenous Equation:** $(D^2 + aD + b)y = 0$
*   **Characteristic Solution for roots D:** $D = \frac{-a \pm \sqrt{a^2 - 4b}}{2}$

| Value of $D$             | Homogenous solution |
| :----------------------- | :------------------ |
| $D \in \mathbb{R}$ (one real root) | $y = C_1 e^{Dx}$    |
| $D = m \pm ni$           | $y = e^{mx} (C_1 \sin nx + C_2 \cos nx)$ |
| $D = R, R$ (repeated roots) | $y = C_1 e^{Rx} + C_2 x e^{Rx}$ |
| $D = R, R, R$ (repeated roots) | $y = C_1 e^{Rx} + C_2 x e^{Rx} + C_3 x^2 e^{Rx}$ |

### Particular Solution, $y_p$
**Method of Undetermined Coefficients**

| $R(x)$                 | $Y_p$ Form                                   |
| :--------------------- | :------------------------------------------- |
| $k$ (constant)         | $y_p = A$                                    |
| Polynomial, $P(x)$     | $y_p = A x^n + B x^{n-1} + \dots + k$      |
| $P(x)\sin(mx)$         | $y_p = (A x^n + \dots + k)\sin mx + (A' x^n + \dots + k')\cos mx$ |
| $e^{mx}P(x)$           | $y_p = e^{mx}(A x^n + B x^{n-1} + \dots + k)$ |

# ENGINEERING ECONOMY

## INTEREST

### Cash Flow
Cash flow is the sum of money recorded as receipts or disbursements in a project's financial records.
*   Expenses - down arrows
*   Receipts - up arrows

### Simple Interest
*   $F = P(1+in)$
*   $F = P + Pin$
*   $I = Pin$

**Types of Simple Interest:**
*   **Ordinary Simple Interest:** Interest period = 360 days
*   **Exact Simple Interest:** Interest period = 365 days

### Compound Interest
$$F = P\left(1 + \frac{r}{m}\right)^{mt}$$
where:
*   $P$ = principal (present worth)
*   $F$ = accumulated amount (future worth)
*   $i$ = rate of interest per interest period = $r/m$
*   $n$ = number of interest periods (total)
*   $m$ = number of interest periods per year

### Continuous Interest
$$F = Pe^{rn}$$
*   $r$ = rate of interest

### Rate of Interest
1.  **Nominal rate of interest rate:** Specifies the rate of interest and the number of interest periods in a year.
2.  **Effective rate of interest:**
    *   **Compound:** $i_{eff} = \left(1 + \frac{r}{m}\right)^m - 1$
    *   **Continuous:** $i_{eff} = e^r - 1$

## Annuity & Gradient

### Ordinary Annuity
*   **Present Worth ($P$):** $P = A \left[ \frac{(1+i)^n - 1}{i(1+i)^n} \right] = A(P/A, i, n)$
*   **Future Worth ($F$):** $F = A \left[ \frac{(1+i)^n - 1}{i} \right] = A(F/A, i, n)$

### Deferred Annuity
*   $P = A \left[ \frac{(1+i)^n - 1}{i(1+i)^n} \right] (1+i)^{-(k-1)}$

## DEPRECIATION

### Uniform Methods

#### A. Straight Line
*   $d = \frac{FC - SV}{L}$
*   $d_{nT} = nd = n \left(\frac{FC - SV}{L}\right)$
*   $BV = FC - d_{nT}$

#### B. Sinking Fund
*   $d_n = \frac{(FC - SV)i}{(1+i)^n - 1}$
*   $d_{nT} = (FC - SV) \left[ \frac{(1+i)^n - 1}{(1+i)^L - 1} \right]$
*   $BV = FC - d_{nT}$

### Non-Uniform Methods

#### A. Declining Balance:
*   $d_{nT} = FC(1 - f)^{L-1} \cdot f$
*   $BV = FC(1 - f)^n$
*   $f = 1 - \sqrt[L]{\frac{SV}{FC}}$

#### B. Double Declining Balance
*   $d_{nT} = FC\left(1 - \frac{2}{L}\right)^{n-1} \cdot \frac{2}{L}$
*   $BV = FC\left(1 - \frac{2}{L}\right)^n$

#### C. Sum of Years Digit
*   $\sum = \frac{L(L + 1)}{2}$
*   **For nth year depreciation ($d_n$):** $d_n = (FC - SV) \frac{L - n + 1}{\sum}$
*   **Total depreciation to nth year ($d_{nT}$):** $d_{nT} = \sum_{1}^{n} (FC - SV) \frac{L - n + 1}{\sum}$
*   **Book Value ($BV$):** $BV = FC - d_{nT}$

## BREAK-EVEN ANALYSIS

$$BEP = \frac{FC}{SP - VC}$$
*   **FC** = Fixed/Overhead cost
*   **SP** = Selling Price
*   **VC** = Variable cost

## COMPARING ECONOMIC ALTERNATIVES

### Equivalent Uniform Annual Cost (EUAC)
$$EUAC = (FC - SV (1+i)^{-L}) \div \sum_{1}^{L} ((1+i)^{-x}) + OHM$$

### Capitalized Cost (CC)
$$CC = FC + \frac{OHM}{i} + \frac{RC - SV}{(1+i)^L - 1}$$

### Financial Ratios
*   **Benefit - Cost Ratio (BCR):** $BCR = \frac{AW (\text{Annual Benefits})}{EUAC}$
*   **Acid - Test Ratio (ATR):** $ATR = \frac{C + CE + MS + AR}{L}$
    *   $C$ = cash
    *   $CE$ = Cash Equivalents
    *   $MS$ = marketable securities
    *   $AR$ = account receivables
    *   $L$ = liabilities
